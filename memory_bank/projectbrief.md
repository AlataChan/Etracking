# 项目概述：订单PDF安全下载工具

## 项目目标
开发一个自动化工具，实现对泰国海关e-Tracking系统的操作自动化，包括登录、订单查询、收据下载并转为PDF格式保存。

## 核心需求
- 自动读取Excel中的提单号
- 自动登录目标系统或复用已登录会话
- 输入订单号并触发收据打印
- 将弹出的收据页面转换为PDF
- 使用提单号命名并保存PDF文件
- 提供异常处理和日志记录
- 模拟人类行为以避免被系统识别为机器人

## 项目范围
- 目标网站：https://e-tracking.customs.go.th/ETS/
- 主要功能：自动化下载订单收据PDF
- 输出：以提单号命名的PDF文件
- 用户：报关、贸易、财务等需要批量下载订单收据的人员

## 技术方向
使用Python和Playwright实现浏览器自动化，通过模拟人类行为和会话管理提高安全性。

## 价值主张
- 提升效率5-10倍
- 标准化输出，命名规范一致
- 增强安全性，避免账号频繁登录
- 适合企业流程自动化集成
