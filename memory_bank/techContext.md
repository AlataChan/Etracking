# 技术上下文

## 开发环境

| 项目 | 规格 |
|------|------|
| 语言 | Python 3.9+ |
| 操作系统 | 跨平台支持（Windows/macOS/Linux） |
| 浏览器 | Chromium (通过Playwright) |
| 内存要求 | 最低4GB，推荐8GB |
| 存储空间 | 最低500MB (不含PDF存储) |

## 核心依赖库

| 库名称 | 版本 | 用途 |
|--------|------|------|
| playwright | 1.30+ | 浏览器自动化 |
| pandas | 2.0+ | Excel数据处理 |
| loguru | 0.6+ | 结构化日志记录 |
| pyyaml | 6.0+ | 配置文件解析 |
| pytest | 7.0+ | 单元测试 |
| tqdm | 4.64+ | 进度条显示 |

## 关键技术组件

### 1. Playwright
- **核心功能**：浏览器自动化、页面操作、元素定位
- **关键特性**：
  - 状态持久化（保存/加载会话）
  - 多页面管理（处理弹窗）
  - 网络请求拦截和监控
  - PDF导出功能
  - 元素等待策略

### 2. Pandas
- **核心功能**：Excel文件读取、数据处理
- **关键特性**：
  - 多Sheet处理
  - 列数据提取
  - 数据过滤和验证

### 3. Loguru
- **核心功能**：日志记录
- **关键特性**：
  - 分级日志
  - 文件轮转
  - 格式化输出
  - 异常捕获和记录

## 技术约束

### 浏览器限制
- 必须使用非Headless模式（网站可能检测Headless浏览器）
- 浏览器窗口不能最小化（影响PDF导出）
- 必须使用真实用户代理（UA）标识

### 网站交互限制
- 请求频率限制：每次操作间隔不少于5秒
- 会话超时：约30分钟（需定期刷新）
- 可能存在验证码（目前未遇到，但需预留处理机制）
- DOM变更：网站更新可能导致选择器失效

### 数据处理限制
- Excel文件格式必须为.xlsx或.xls
- 提单号必须位于F列
- 每个Sheet都会被处理
- 空行或格式不正确的提单号将被跳过

## 开发工具链

- **IDE**：VSCode 或 PyCharm（配置Playwright插件）
- **版本控制**：Git
- **依赖管理**：pip/venv/requirements.txt
- **测试**：pytest
- **文档**：Markdown
- **本地开发流程**：
  1. 配置Python虚拟环境
  2. 安装依赖 (`pip install -r requirements.txt`)
  3. 安装浏览器 (`playwright install`)
  4. 配置 `settings.yaml`
  5. 运行 `main.py`

## 部署注意事项

- 部署环境需要图形界面支持（XServer/VNC等）
- 必须安装浏览器依赖库
- 必须有文件系统写入权限（PDF保存）
- 建议使用Python虚拟环境隔离依赖
- 避免使用共享网络（可能影响请求速度和稳定性）
